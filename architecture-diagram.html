Bookmark Manager Pro 系统架构图

Manifest V3 浏览器扩展：Popup UI、Background Service Worker、Chrome/Edge API、Storage 与构建依赖关系。

Runtime: Chrome / Edge MV3
Build: Webpack + CopyWebpackPlugin

运行时主链路

  • Popup UI 负责状态、搜索、渲染和用户操作。
  • 后台 Service Worker 通过消息路由处理书签、标签、导入导出。
  • Chrome/Edge APIs 是唯一外部运行时依赖。

数据与缓存

  • bookmarks API 提供书签树、搜索与 CRUD。
  • storage.local 保存标签库、缓存书签、搜索语言等本地状态。
  • storage.sync 保存可同步的设置项。

构建与交付

  • Webpack 打包 popup.js 与 background.js。
  • CopyWebpackPlugin 复制 manifest、HTML、CSS 与图标。
  • dist/ 作为 Chrome/Edge unpacked extension 加载目录。
